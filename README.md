# CSV Cleaner – Automatización de Limpieza de Datos

Este proyecto es un script en Python que automatiza la limpieza de archivos CSV.  
Elimina duplicados, estandariza texto y ordena los datos por una columna específica.

## Funcionalidades
- Eliminación de filas duplicadas  
- Normalización de texto (minúsculas, espacios)  
- Ordenamiento por columna  
- Generación de un archivo limpio listo para usar  

## Uso
```python
limpiar_csv("datos_originales.csv", "datos_limpios.csv", "nombre")
